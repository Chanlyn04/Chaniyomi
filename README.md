# Chaniyomi
Chaniyomi Webtoon App.

# Features
* Share Projects
* Message other developers
* Rate others work
* Search other developers

# Course Overview
* Basic React Native Components
* Database Design using Appwrite
* User Registration & Authenticaiton
* Deployment

# Tech Stack
* React Native
* AppWrite
* HTML, CSS and JS
